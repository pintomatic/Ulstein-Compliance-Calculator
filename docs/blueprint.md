# **App Name**: ETS Compliance Navigator

## Core Features:

- Hero Section: Displays a concise hero section presenting the core problem (EU ETS fines) and the Ulstein Digital solution (data-driven compliance).
- Regulatory Timeline: Presents a timeline of key regulatory milestones (CII, EU ETS, FuelEU) to contextualize the urgency of compliance.
- Interactive ETS Cost Slider: Interactive slider allows users to estimate their potential EU ETS costs based on CO₂ emissions, providing a tangible financial impact assessment.
- ROI Calculator: ROI calculator to project potential savings from fuel efficiency improvements facilitated by Ulstein Digital's solutions.
- Case Study Carousel: Carousel of case studies demonstrating quantifiable fuel and time savings achieved by Ulstein Digital's clients.
- Visual Guide: Visual representation of the installation of a Blue Box using Nextjs.
- Personal EU ETS Summary: An AI powered "tool" summarizes EU ETS changes for individual ship details.

## Style Guidelines:

- Primary color: Fjord Blue (#1789C6) reflects the maritime heritage and reliability of Ulstein.
- Background color: Glacier White (#F8F9FA) provides a clean, neutral backdrop for content.
- Accent color: Safety Orange (#FF6600) draws attention to key CTAs and highlights, signifying action and urgency.
- Body and headline font: 'Inter' (sans-serif) for a clean, metric-driven Nordic aesthetic, providing excellent readability and a modern feel.
- Simple, geometric icons to represent key features and benefits, ensuring clarity and ease of understanding.
- Single-scroll layout guides the user through a structured narrative, emphasizing key information and calls to action.
- Subtle count-up animations for key metrics, adding visual interest and highlighting the quantifiable impact of Ulstein Digital's solutions.
